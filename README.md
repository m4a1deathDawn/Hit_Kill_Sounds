# 命中/击杀音效 (Hit/Kill Sounds)

一个暗潮 (Darktide) MOD，可以在命中或击杀敌人时播放来自不同游戏的命中和击杀音效，并显示动态击杀图标。

## 功能特点

- **命中音效**：命中敌人时播放不同音效（普通命中 vs 弱点命中）
- **击杀音效**：击杀敌人时播放不同音效（普通击杀 vs 爆头击杀）
- **动态击杀图标**：队列式击杀图标系统，支持连续击杀图标累积
- **随机选择**：每种类型有多个音效变体，随机选择播放
- **音效冷却**：防止快速射击武器导致音效过载
- **去重机制**：在多人模式下防止重复播放音效
- **DoT 支持**：可选的持续伤害命中音效（流血、燃烧等）
- **目标过滤**：可设置音效仅对特定敌人类型生效（精英、专家、Boss）
- **音量控制**：可独立调节命中和击杀音效音量（0-100）
- **图标自定义**：可调节图标大小、颜色、水平/垂直位置、显示时长
- **多人模式支持**：在单人模式和多人模式下均能正常工作
- **CF 连杀机制**（v1.1）：穿越火线风格的连杀音+图标系统，按连杀计数顺序播放音效，可切换风格
- **风格切换**（v1.1）：可在「战地 5 队列式图标」和「穿越火线连杀式图标」之间切换
- **CF 音/图独立**（v1.1）：穿越火线连杀音效可与图标风格独立组合，支持 4 种组合（CF 音+BF5 图、BF5 音+CF 图、CF+CF、BF5+BF5）

## 击杀图标

v0.9 大改版！

- **队列系统**：最多同时显示10个击杀图标，新击杀推动旧图标左移
- **入场动画**：图标从屏幕右侧滑入到中央（ease-out, 0.3s）
- **普通击杀**：入场 scale 1.8→1.0（ease-out），停留后淡出
- **爆头击杀**：同普通击杀 + 圆环冲击波效果（scale 1→3, 透明度衰减）
- **离场动画**：向左滑出并淡出（ease-out, 0.2s）
- **可调参数**：
  - 图标开关
  - 图标大小（0.5x - 2.0x）
  - 图标透明度（0% - 100%）
  - 普通图标颜色（RGB）
  - 爆头图标颜色（RGB）
  - 水平位置（0=最左，50=居中，100=最右）
  - 垂直位置（0=顶部，100=底部）
  - 显示时长（1.0s - 3.0s）

图标资源：战地 V 击杀图标

## CF（穿越火线）连杀机制

v1.1 新增！从 EBuyToDeep_KillFeedBack MOD 移植的穿越火线主题连杀系统，与默认的「战地 5 队列式图标」互补。

- **连杀音效**：按 `killsound_cf_01..09.wav` 顺序播放（1→2→…→9→循环），区别于普通击杀音的「随机播放」
- **连杀图标**：`kill1..6.png` 逐级显示 + 首杀爆头用专属金色 `headshot_gold.png`
- **Boss 专属音**：击杀 monster 标签敌人播放 `killsound_cf_boss.wav`
- **风格切换**：在「击杀图标设置」顶部"风格"下拉框中选择「穿越火线风格」即可启用
- **可调参数**：
  - 连杀计数上限（10-30，默认 13，达到上限后 wrap-around 归零重新开始）
  - 图标透明度、大小、垂直/水平位置
  - 计数器重置时间（1.0s-3.0s，默认 2.0s，同时控制连杀窗口和图标显示时长）
- **CF 音/图独立**：在「击杀音效设置」中可独立启用"CF 击杀音效（连杀）"开关，与图标风格解耦——4 种组合（CF 音+BF5 图、BF5 音+CF 图、CF+CF、BF5+BF5）均支持

## 工作原理

本 MOD 通过 Hook `AttackReportManager:add_attack_result` 捕获攻击事件：

- 拦截玩家的伤害/击杀事件
- 判断是命中（普通/弱点）还是击杀（普通/爆头）
- 根据敌人类型应用目标过滤
- 通过外部音频播放器播放相应的音效
- 击杀时调用 HUD 系统显示击杀图标

外部音频播放器 (HitKillSoundsPlayer) 通过 HTTP 请求处理音效播放，支持多通道音频（命中和击杀音效使用独立通道）。

击杀图标系统预加载纹理，通过队列管理10个 slot，新击杀触发入场动画推动旧图标左移。

## 安装方法

1. 下载 MOD 并将其放入暗潮的 MOD 文件夹中
2. 在游戏的 MOD 菜单中启用该 MOD
3. 在 MOD 设置中配置你喜欢的音效来源和音量，以及击杀图标选项

## 系统要求

- 已启用 MOD 支持的暗潮 (Darktide)
- HitKillSoundsPlayer（随本 MOD 一起提供）

## 音效来源
