# 命中/击杀音效 (Hit/Kill Sounds)

一个暗潮 (Darktide) MOD，可以在命中或击杀敌人时播放来自不同游戏的命中和击杀音效，并显示动态击杀图标。

当前版本：v1.4.2

## 功能特点

- **命中音效**：命中敌人时播放不同音效（普通命中 vs 弱点命中）
- **击杀音效**：击杀敌人时播放不同音效（普通击杀 vs 爆头击杀）
- **动态击杀图标**：队列式击杀图标系统，支持连续击杀图标累积
- **随机选择**：每种类型有多个音效变体，随机选择播放
- **音效冷却**：防止快速射击武器导致音效过载
- **去重机制**：在多人模式下防止重复播放音效
- **DoT 支持**：可选的持续伤害命中音效（流血、燃烧等）
- **目标过滤**：可设置音效仅对特定敌人类型生效（精英、专家、Boss）
- **音量控制**：可独立调节命中和击杀音效音量（0-100）
- **图标自定义**：可调节图标大小、颜色、水平/垂直位置、显示时长
- **多人模式支持**：在单人模式和多人模式下均能正常工作
- **CF 风格连杀输出**（v1.1）：穿越火线风格的连杀音+图标系统，按连杀计数顺序播放音效，可切换风格
- **风格切换**（v1.1）：可在「战地 5 队列式图标」和「穿越火线连杀式图标」之间切换
- **CF 音/图独立**（v1.1）：穿越火线连杀音效可与图标风格独立组合，支持 4 种组合（CF 音+BF5 图、BF5 音+CF 图、CF+CF、BF5+BF5）
- **SimpleAudio + SimpleAssets 后端**（v1.2）：音频默认通过 SimpleAudio 播放，图标默认通过 SimpleAssets 加载，不再依赖外部播放器可执行文件
- **CODBO7 击杀音效**（v1.3）：普通击杀从 3 个变体随机播放，爆头击杀使用专用音效；CODBO7 只出现在击杀来源下拉框
- **哀星号自定义 BGM**（v1.3）：可选在普通 `hub` 大厅随机播放 6 首本地 MP3，并独立调节音量
- **通用连杀机制与高频播放保护**（v1.35）：BF4 文字、CF 音效和 CF 图标分别维护独立连杀计数；SimpleAudio 使用有界队列、分帧调度和多实例 voice pool
- **战地4风格文字击杀信息与独立计分**（v1.4）：使用固定槽位显示击杀事件和累计分值，独立于 BF5/CF 音效与图标开关
- **独立输出计数**（v1.4.2）：BF4、CF 音效、CF 图标各自使用独立计数器，筛选条件不同时三个连杀序号可以不同

## 击杀图标

  - 图标大小（0.5x - 2.0x）

  - 普通图标颜色（RGB）
  - 爆头图标颜色（RGB）
  - 水平位置（0=最左，50=居中，100=最右）
  - 垂直位置（0=顶部，100=底部）
  - 显示时长（1.0s - 3.0s）

## CF（穿越火线）连杀机制

v1.1 新增！从 EBuyToDeep_KillFeedBack MOD 移植的穿越火线主题连杀系统，与默认的「战地 5 队列式图标」互补。

- **连杀音效**：按 `killsound_cf_01..09.wav` 顺序播放（1→2→…→9→循环），区别于普通击杀音的「随机播放」
- **连杀图标**：`kill1..6.png` 逐级显示 + 首杀爆头用专属金色 `headshot_gold.png`
- **Boss 专属音**：击杀 monster 标签敌人播放 `killsound_cf_boss.wav`
- **风格切换**：在「击杀图标设置」顶部"风格"下拉框中选择「穿越火线风格」即可启用
- **可调参数**：
  - 连杀计数上限（10-30，默认 13，三个计数器分别循环）
  - BF4 文字目标（`bf4_feed_target`）、CF 音效目标（`kill_target`）、CF 图标目标（`kill_icon_target`）
  - 图标透明度、大小、垂直/水平位置
  - 计数器重置时间（1.0s-3.0s，默认 2.0s，同时控制连杀窗口和图标显示时长）
- **CF 音/图独立**：在「击杀音效设置」中可独立启用"CF 击杀音效（连杀）"开关，与图标风格解耦——4 种组合（CF 音+BF5 图、BF5 音+CF 图、CF+CF、BF5+BF5）均支持
- **筛选语义独立**：`bf4_feed_target` 只决定 BF4 文字事件，`kill_target` 只决定 CF/BF5 击杀音效，`kill_icon_target` 只决定 CF/BF5 击杀图标；`killstreak_target` 已移除，旧配置值会被忽略

图标资源：穿越火线连杀图标

## 战地4风格文字击杀信息（v1.4）

在「战地4风格文字击杀信息」设置组中打开 `bf4_feed_enabled`，即可显示无面板、无边框的原生 HUD 文字击杀信息。该功能默认关闭，并且必须先在「通用设置」中启用 `killstreak_enabled`。

- 普通击杀 100、爆头奖励 50、精英击杀 200、专家击杀 250、Boss 击杀 500
- 同一次爆头特殊击杀显示「爆头奖励」和「击杀专家」两行，累计分值一次性增加 300
- 最新事件位于底部，事件文本右对齐、分值左对齐；累计分值固定显示在信息列右上方并带短促缩放动画
- 支持目标过滤、显示时长（1.0-3.0 秒）、水平/垂直位置和文字缩放（50%-150%）
- 该功能不依赖 CF/BF5 音效或图标开关，也不会触发官方 Combat Feed 事件；如果同时开启游戏原生击杀信息，画面可能出现重复内容
- 关闭文字信息或连杀机制会立即清空文字槽位和累计分值，不会修改已保存的设置

## 工作原理

本 MOD 通过 Hook `AttackReportManager:add_attack_result` 捕获攻击事件：

- 拦截玩家的伤害/击杀事件
- 判断是命中（普通/弱点）还是击杀（普通/爆头）
- 根据敌人类型应用目标过滤
- 通过 SimpleAudio 播放相应的本地音效
- 击杀时调用 HUD 系统显示击杀图标
- 有效击杀另外提交一次 BF4 文字信息批次，独立维护累计分值，不修改官方 Combat Feed

v1.2 起，音频播放默认使用 SimpleAudio，图标纹理默认使用 SimpleAssets。MOD 仍保留旧 HTTP 播放器适配层作为兼容 fallback，但发布到 N 网的版本可以不包含 `bin` 目录。

击杀图标系统通过 SimpleAssets 预加载纹理，通过队列管理10个 slot，新击杀触发入场动画推动旧图标左移。

## 哀星号自定义 BGM（v1.3）

在「哀星号自定义 BGM」设置组中打开 `lobby_bgm_enabled` 后，MOD 只在
`Managers.state.game_mode:game_mode_name() == "hub"` 的普通哀星号大厅播放
`audio/BGM/Lobby_BGM1.mp3` 至 `Lobby_BGM6.mp3`。进入大厅时随机选曲，曲目结束后从其他曲目中随机选择，避免连续重复。

- 默认关闭；不作用于任务、主菜单、训练场、射击场或 `prologue_hub`
- `lobby_bgm_volume` 范围为 0-100，步长 5；播放中修改音量会直接更新当前播放实例，不会从头重播
- 依赖 **SimpleAudio** 的 `play_file`，使用 `audio_type = "music"`，同时遵循游戏主音量和音乐音量
- 只有在自定义曲目成功取得专属 `play_id` 后才尝试接管大厅原生音乐；如果 SimpleAudio、文件或播放初始化失败，会保留原生大厅音乐

## 安装方法

1. 安装并启用前置 MOD：SimpleAudio、SimpleAssets
2. 下载本 MOD，并将 `Hit_Kill_Sounds` 与 `SimpleAudio`、`SimpleAssets` 平级放入暗潮的 `mods` 文件夹中
3. 在游戏的 MOD 菜单中启用该 MOD
4. 在 MOD 设置中配置你喜欢的音效来源和音量、击杀图标选项；如需自定义大厅音乐，再打开「哀星号自定义 BGM」
